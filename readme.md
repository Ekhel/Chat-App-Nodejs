* How to Use this project?
  - clone this Repo 
    ```
    https://github.com/Ekhel/Chat-App-Nodejs.git
    ```

  - Node Initialize (Install dependencies)
    ```javascript
    npm install
    ```

  - Run App (Dev)
    ``` 
    npm run dev
    ```

  - Run App (Deploy Production)
    ```
    npm run start
    ```

  - Specify a room name
